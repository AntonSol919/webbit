An example page

This is an example page. There is nothing interesting here. Add 'uploader' to the search paramter.

Once you see the blue bar, you should edit the destination path.

Next you can use the 'Edit' checkbox to edit this text, or use F12 to use the browser tools.

You can do the ?uploader trick for any html page on webbit. Find a page you like and add 'uploader' to the query string to steal copy its design.

Once you're satified - press upload. You'll be taken to quarantine. Here you input/generate your identity to vouch for the content you're uploading.

You can also add javascript. That sounds dangerous, but its not more dangerous than any other unkown website you find with google. Javascript is sandboxed to this subdomain, and there are no cookies or identities to steal from webbit.

Javascript allows us to get meta. We can for instance create a forum thread with comments:/example.forum.editor.html.

Think the editor needs better styling? Want to support nested comments? - just add '?uploader' and build it. (or send a POST request to the destination url ending with ?data)
